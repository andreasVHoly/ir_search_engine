The Thesaurus functionality in the seach engine uses a Natural Language Toolkit (NLTK).
To install the NLTK on your computer, you need pip installed.
The following steps are to install pip and then NLTK:
	1) Install pip: sudo apt-get install python3-pip
	2) Install nltk: sudo pip3 install -U nltk
	3) Install numpy: sudo pip3 install -U numpy
	4) Download the wordnet package: 
		4.1) $ python3
		4.2) import nltk
		4.3) nltk.download('wordnet')
