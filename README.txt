The Thesaurus functionality in the seach engine uses a Natural Language Toolkit (NLTK).
To install the NLTK on your computer, you need pip installed.
The following steps are to install pip and then NLTK:
	1) In the directory of get-pip.py run the command line "python get-pip.py --user".
	2) Then "sudo apt-get install python-pip"
	3) And then to install NLTK, "pip install nltk"
